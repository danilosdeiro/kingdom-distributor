# MeuKingdom APK

Arquivo recomendado para compartilhar:

`client-react/android/app/build/outputs/apk/release/MeuKingdom-1.5.apk`

Mensagem curta para mandar no grupo:

```text
Pessoal, segue o app MeuKingdom para organizar a partida.

Como instalar no Android:
1. Baixe o arquivo MeuKingdom-1.5.apk.
2. Toque no arquivo baixado.
3. Se o Android avisar que o app veio de fora da Play Store, toque em Configuracoes e permita instalar desta fonte.
4. Volte e toque em Instalar.

Depois de instalar, podem entrar pelo app ou pelo site:
https://meukingdom.vercel.app
```

Observacoes:

- O aviso do Android aparece porque o APK nao esta publicado na Play Store.
- O app usa o mesmo servidor do site.
- Links de convite do site `https://meukingdom.vercel.app/CODIGO` podem abrir direto no app quando o Android reconhecer o vinculo do dominio.
